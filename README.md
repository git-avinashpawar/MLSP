
# MLSP: Machine Learning for Signal Processing

ENGR-E 511: Machine Learning for Signal Processing
Spring 2022, Indiana University

## Overview

This repository contains course materials, assignments, and resources for ENGR-E 511, focused on machine learning applications in signal processing.[^1]

## Repository Structure

- `/notebooks` – Jupyter Notebooks containing lecture notes, demonstrations, and assignments.[^1]
- `/html` – HTML files generated from Jupyter Notebooks, for easy viewing in a browser.[^1]


## Technologies

- Python (Jupyter Notebook)
- HTML


## Contents

- Lecture notes and practical demos
- Signal processing datasets and examples
- Machine learning algorithms in Python
- Assignments and solutions


## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/git-avinashpawar/MLSP.git
```

2. Install Python and Jupyter Notebook.
3. Open notebooks in JupyterLab or Jupyter Notebook.
